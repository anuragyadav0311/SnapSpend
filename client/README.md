# Expense Tracker Frontend

## Setup

```bash
cd client
npm install
cp .env.example .env
```

Set `VITE_API_URL` in `.env` to the running backend URL.

## Run

```bash
cd client
npm run dev
```

## Build

```bash
cd client
npm run build
```

## Available Pages

- `/login`
- `/register`
- `/dashboard`
- `/expenses`
